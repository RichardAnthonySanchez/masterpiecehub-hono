# Masterpiece Hub

Masterpiece Hub is an online platform that curates art pieces from various historical periods, artists, and art movements.  
Built with a modern full-stack setup using **Node.js**, **TypeScript**, **Hono**, **MongoDB**, and **React/Vite**.
